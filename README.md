# Sports Mini Calendar 2026

2026년 주요 스포츠 경기 일정을 월간 캘린더로 빠르게 확인하는 정적 웹사이트입니다.

## 사용 방법

- `index.html`을 브라우저로 열면 바로 실행됩니다.
- `script.js`의 `ufcEvents2026` 배열에서 UFC 날짜, 시간, 대회명, 장소, 링크를 수정할 수 있습니다.
- UFC 일정은 2026년 5월 31일 기준으로 정리했습니다. 공개 JSON/ICS API가 확인되지 않아 로컬 데이터로 관리합니다.
- 프리미어리그와 라리가는 football-data.co.uk 공개 CSV를 자동으로 불러와 2026년 경기만 달력에 표시합니다.
- 브라우저가 외부 CSV 요청을 막는 경우에는 `script.js`의 `footballFixtures2026` 로컬 일정이 표시됩니다.
- 상단 `Text` 슬라이더로 글자 크기를 조절할 수 있습니다.
- 캘린더 영역에 이미지를 드래그 앤 드롭하면 상단 배경 이미지로 저장됩니다.
- 외부 일정/중계 링크는 새 탭으로 열립니다.
